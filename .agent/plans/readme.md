# Plans & Implementation History

Past PRDs and implementation records. Reference these before implementing similar features.

## How to add a plan

After completing a feature, create a subfolder:
```
plans/
└── feature-name/
    ├── prd.md          ← Requirements and design
    └── implementation.md ← What was actually built, deviations, learnings
```
